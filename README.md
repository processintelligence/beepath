# BeePath

This is BeePath, a tool that allows you to quickly design models starting from natural language.

The original code of the conversion has been created by Ana Maria Sima and Antonio Grama for their master thesis entitled "[Optimization of natural language descriptions of business processes](https://findit.dtu.dk/en/catalog/67fef56f9e134e0a88e14877)" and its source code is available at https://github.com/Toni751/Thesis-WebApp.
